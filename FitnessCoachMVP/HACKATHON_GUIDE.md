## 🎨 UI/UX Hackathon Hacks

### Make It Look Expensive
- Use gradient backgrounds (not flat colors)
- Add subtle shadows and depth
- Use consistent spacing
- Add smooth animations
- Make buttons interactive

### Apple Fitness+ Inspiration
```css
/* Glassmorphism effect */
.card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 20px;
}

/* Gradient text */
h1 {
  background: linear-gradient(135deg, #667eea, #764ba2);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

/* Smooth animations */
button {
  transition: all 0.3s ease;
}
button:active {
  transform: scale(0.95);
}
```

### Dark Mode is Popular
- Use dark background with light text
- Add subtle colors
- Makes AI responses feel "futuristic"

## 🚩 Common Mistakes to Avoid

❌ **DON'T:**
- Show hardcoded fake data (especially when you have real AI!)
- Make slow requests visible (test everything locally first)
- Have backend crashes during demo
- Use generic, non-personalized responses
- Show unfinished UI

✅ **DO:**
- Pre-load some data in the API
- Use a local backend (not making HTTP requests)
- Have a backup demo video ready
- Show examples of DIFFERENT personalized responses
- Polish the UI with Figma/CSS before demo
